# 🚀 Shutly

> You said “just a few minutes.”
> Shutly remembers exactly how many.

Shutly is a lightweight shutdown scheduler built for one simple reason:

**sometimes you need your PC to enforce the limit for you.**

No bloat. No unnecessary features. Just set a timer and let it do its job.

---

## ✨ Features

* ⏱ Set a timer (hours / minutes / seconds)
* 🔌 Shutdown or Restart your PC
* ⏸ Pause / Resume anytime
* ❌ Cancel instantly
* 🧃 Runs quietly in the system tray
* ⚡ Lightweight and fast

---

## 🧠 Why Shutly exists

Ever lent your laptop and heard:

> “I’ll give it back in a bit”

…and that “bit” turned into an entire movie?

Yeah.

Shutly doesn’t argue.
It doesn’t remind.
It just **acts when the time is up.**
